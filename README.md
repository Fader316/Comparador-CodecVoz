# Comparador-CodecVoz

## Descripción
Aplicación web interactiva para comparar técnicas de codificación de voz (LPC, RELP y CELP),
mostrando métricas clave, simulación gráfica y un desafío educativo tipo juego.
Version 2.0

## Desarrolladores

## Tecnologías
- HTML5
- CSS3 (Responsive, Flexbox, Grid)
- JavaScript (Canvas API)

## Uso Web
1. Descomprime el archivo.
2. Abre `index.html` en cualquier navegador moderno.

## Instalación como APK (Android)
Esta aplicación puede instalarse como APK usando un contenedor WebView.

### Opción recomendada
1. Usar **Android Studio**.
2. Crear un proyecto *Empty Activity*.
3. Copiar `index.html` , `style.css` y  `script.js` a la carpeta `assets/`.
4. Habilitar WebView y cargar el archivo local.
5. Compilar y generar el APK.

### Opción rápida
- Usar herramientas como **WebViewGold**, **PWA Builder** o **APK Editor** para empaquetar la web como APK.
